# logseq-claude-plugin

Claude Code plugin for Logseq knowledge management and plugin development.

## Features

### Skills Included

1. **logseq-knowledge-management** - Master Logseq for PKM workflows
   - Graph queries with Datascript/Datalog
   - Block and page organization
   - Properties and tags
   - Daily notes and Zettelkasten patterns
   - Task management

2. **logseq-plugin-development-api** - Develop Logseq plugins with @logseq/libs
   - Plugin structure and lifecycle
   - Editor, DB, App, and UI APIs
   - Datascript queries
   - Custom commands and UI
   - DB version changes (2025)

## Installation

This plugin is available through the ai-skills marketplace:

```bash
# Add the marketplace (if not already added)
claude plugin marketplace add briansunter/ai-skills

# Install the plugin
claude plugin install logseq-claude-plugin@personal-plugins
```

Or install directly from GitHub:

```bash
claude plugin install briansunter/logseq-claude-plugin
```

## Usage

The skills auto-activate when working with Logseq-related tasks:

- Setting up or using Logseq for PKM
- Writing Datascript/Datalog queries
- Developing Logseq plugins
- Working with @logseq/libs APIs
- Managing blocks, pages, and properties
